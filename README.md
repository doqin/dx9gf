# directx9-game-framework

Đây là một dự án dành cho học môn Nhập môn lập trình game - SE102. Để tạo một framework sử dụng các API của Direct3D trong DirectX 9.

Vì đây là một dự án sử dụng DirectX 9 nên chúng mình phải tải SDK legacy của Microsoft về để có thể sử dụng được các API của DirectX 9. Link tải SDK legacy: https://www.microsoft.com/en-us/download/details.aspx?id=8109
